# 🎮 Color Switch Game

A fast-paced **Color Switch–style arcade game** built using **JavaScript** and **HTML5 Canvas**.  
The objective is simple: **match colors, avoid wrong blocks, and score as high as possible**.

🔗 **Live Demo:**  
https://technogirdher.github.io/color-switch-game/

🔗 **GitHub Repository:**  
https://github.com/technogirdher/color-switch-game

---

## 🚀 Features
- Smooth **gravity-based jumping**xdxdxxdx
- **Dynamic rotating obstacles**
- Increasing difficulty as score increases
- **Keyboard + Mobile touch support**
- Real-time score tracking
- Fully deployed using **GitHub Pages**

---

## 🎮 Controls
### Desktop
- **Spacebar** → Jump

### Mobile
- **Tap anywhere on the screen** → Jump

---

## 🛠️ Tech Stack

- **HTML5 Canvas**
